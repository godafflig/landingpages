# ALDRET Landing Page

A responsive landing page for ALDRET, a marketplace platform for sports and mountain equipment.

## Features

- **Hero Section**: Eye-catching hero with call-to-action buttons for buying and selling
- **Categories**: Browse winter and summer sports equipment
- **Why Choose ALDRET**: Four key features highlighting security, efficiency, quality, and pricing
- **Security Section**: Emphasizes secure payments and verified sellers
- **Selling Steps**: Step-by-step guide for sellers
- **Questionnaire Section**: Call-to-action for user feedback
- **Footer**: Complete footer with navigation, legal links, and social media

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Technology Stack

- React 18
- Vite
- CSS3
- Responsive Design

## Design

The component faithfully recreates the Figma design with:
- Exact color matching
- Proper font usage (Helvetica, Kdam Thmor Pro, Inter)
- Responsive breakpoints for mobile, tablet, and desktop
- Semantic HTML structure
- CSS classes instead of inline styles
- Accessible markup with proper ARIA labels
